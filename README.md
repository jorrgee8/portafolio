# Portafolio Profesional — Jorge

Este es un portafolio web personal y minimalista, diseñado para presentar un perfil profesional enfocado en el desarrollo de software multiplataforma. El diseño prioriza la la limpieza visual y una experiencia de usuario fluida.

## 🚀 Características

* **Diseño Minimalista y Elegante:** Paleta de colores profesional y sencilla
* **Totalmente Responsivo:** Adaptado para una visualización óptima en cualquier dispositivo (móviles, tablets y ordenadores).
* **Navegación Fluida:** Desplazamiento fluido entre las diferentes secciones a través del menú de navegación superior.
* **Menú de Navegación Activo (Intersection Observer):** El navbar detecta automáticamente en qué sección se encuentra el usuario y resalta el botón correspondiente de forma dinámica.
* **Efecto de Entrada Suave:** Animación fluida al hacer scroll que desplaza las secciones suavemente hacia arriba para una experiencia más orgánica.
* **Sin Dependencias Externas:** Construido de forma nativa utilizando exclusivamente HTML5, CSS3 y JavaScript.

## 📁 Estructura del Proyecto

El proyecto está compuesto por tres archivos principales organizados de manera limpia y eficiente:

```text
├── index.html      # Estructura semántica del sitio y contenido del portafolio.
├── styles.css      # Estilos, maquetación adaptativa, paleta de colores y tipografía.
└── script.js      # Lógica de navegación, scroll suave y observadores de secciones.
